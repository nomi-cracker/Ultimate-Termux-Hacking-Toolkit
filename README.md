# 🚀 Ultimate Hacking Toolkit for Termux & Linux

## 🛠️ Installation
1. Clone the repository or download `install.sh`.
2. Run: `chmod +x install.sh`
3. Run: `./install.sh`

## 💻 Usage
After installation, tools are located in `~/Tools/Hacking_Kit`.
